# interactive target selection

### Web-app

Run live at https://py.cafe/bmorris3/rocky-targets

### Run web-app locally

```bash
git clone https://github.com/bmorris3/rocky-targets
cd solara
pip install -r requirements.txt
solara run app.py
```
